# Support

## Where To Ask

- Bug reports: GitHub Issues
- Feature requests: GitHub Issues
- Usage questions: GitHub Issues with enough context to reproduce the problem

## What To Include

For faster triage, include:

- app version
- operating system
- whether you are using a release build or local dev build
- exact steps to reproduce
- expected result
- actual result
- logs, screenshots, or screen recordings when helpful

## Before Opening an Issue

- check the latest [CHANGELOG.md](./CHANGELOG.md)
- confirm the issue still reproduces on the latest release
- avoid posting secrets, tokens, or private project data

## Maintainer Scope

This project is maintained on a best-effort basis. Support quality and response time may vary depending on issue complexity and maintainer availability.
